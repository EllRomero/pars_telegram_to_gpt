Цель программы:
Парсить n последних постов за сегодняшний день в указанных группах, после чего перефразировать текст через AI.
Программа автоматически считает, сколько постов было проверенно, а так же сравнивает был ли данный пост прочитан программой в этот день.
Все данные сохраняются в созданную папку text. Далее папки будут разбиты по дате и названию группы.

Настриваемые параметры:
-В папке group, текстовый файл group_name. Указывать группы которые нужно парсить, разделяя их запятой.
-В модуле freegpt, можно выбирать модель для gpt (Подробнее https://github.com/xtekky/gpt4free)
